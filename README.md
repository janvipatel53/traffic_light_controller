# Smart 4-Way Traffic Light Controller (Verilog)

This project is a 4-way traffic light controller made using Verilog HDL.  
It is designed using FSM (Finite State Machine) concept and controls traffic lights for North, East, South and West roads.

## Features
- 4-way junction traffic signal control
- FSM based design with proper state transitions
- Gives green light to one road at a time for safety
- Yellow light is used before switching to the next road
- Reset functionality to start the system from initial state
- Works based on car sensor inputs from all four sides
